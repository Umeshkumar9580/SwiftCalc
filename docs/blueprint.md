# **App Name**: SwiftCalc

## Core Features:

- Standard Operations: Perform addition, subtraction, multiplication, and division with a high-precision calculation engine.
- History Log: Store and retrieve past calculations from a local database for quick reference.
- AI Word Problem Solver: An AI tool that uses reasoning to interpret natural language math problems and provide step-by-step solutions.
- Unit Conversion: Easily convert between different units of measurement like length, weight, and temperature.
- Scientific Mode: Access advanced mathematical functions including trigonometry, logarithms, and exponents.

## Style Guidelines:

- Primary background: Deep obsidian (#121212) for a modern dark aesthetic.
- Operator buttons: Vibrant orange (#FF9500) to provide clear visual distinction.
- Display font: 'Roboto Mono' for consistent character spacing in numerical results.
- Interface font: 'Inter' for high legibility on buttons and menu labels.
- Minimalist, thin-line glyphs for history, settings, and mode toggles.
- Responsive grid-based keypad with a glassmorphism-inspired result display.
- Smooth sliding transitions when switching between standard and scientific modes.
